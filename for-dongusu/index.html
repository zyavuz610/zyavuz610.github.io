Programlamada en temel yapı taşlarından biri olan for döngülerini interaktif örnekler ve uygulamalarla öğrenin. Sıfırdan ileri seviyeye kadar kapsamlı eğitim!

For Döngüsü Nedir?

Tekrarlama

For döngüsü, belirli bir kod bloğunu belirlenen sayıda tekrarlamak için kullanılır. Bu sayede aynı işlemi defalarca yazmak zorunda kalmazsınız.

Verimlilik

Kod yazma süresini kısaltır, hata yapma olasılığını azaltır ve kodunuzu daha okunabilir hale getirir.

Kontrol

Döngünün kaç kez çalışacağını, hangi koşullarda duracağını tam olarak kontrol edebilirsiniz.

For Döngüsü Türleri

JavaScript
Klasik For Döngüsü
for (let i = 0; i < 10; i++) {
    console.log("Sayı: " + i);
}

En yaygın kullanılan for döngüsü türüdür. Başlangıç değeri, koşul ve artış miktarı belirtilir.

JavaScript
For...of Döngüsü
const fruits = ["elma", "muz", "portakal"];
for (let fruit of fruits) {
    console.log(fruit);
}

Diziler ve diğer iterable nesneler üzerinde dolaşmak için kullanılır. Daha okunabilir kod sağlar.

Python
Python For Döngüsü
for i in range(10):
    print(f"Sayı: {i}")

# Liste ile
fruits = ["elma", "muz", "portakal"]
for fruit in fruits:
    print(fruit)

Python'da for döngüsü daha basit syntax'a sahiptir ve range() fonksiyonu ile sayı aralıkları oluşturulur.

Java
Java For Döngüsü
// Klasik for
for (int i = 0; i < 10; i++) {
    System.out.println("Sayı: " + i);
}

// Enhanced for (for-each)
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}

Java'da hem klasik for hem de enhanced for (for-each) döngüsü kullanılabilir.

Pratik Örnekler

Sayı Toplama
let toplam = 0;
for (let i = 1; i <= 100; i++) {
    toplam += i;
}
console.log("1-100 arası toplam: " + toplam);

1'den 100'e kadar olan sayıları toplar.

Fibonacci Serisi
let a = 0, b = 1;
console.log(a, b);
for (let i = 2; i < 10; i++) {
    let c = a + b;
    console.log(c);
    a = b;
    b = c;
}

Fibonacci serisinin ilk 10 terimini hesaplar.

Çarpım Tablosu
for (let i = 1; i <= 10; i++) {
    for (let j = 1; j <= 10; j++) {
        console.log(i + " x " + j + " = " + (i*j));
    }
}

İç içe for döngüleri ile çarpım tablosu oluşturur.
